# An explicit non-invertible endomorphism of the third Weyl algebra

**Draft — [Author name], with Claude (Fable 5, Anthropic)**
*July 21, 2026*

## Abstract

Building on L. Alpöge's announced counterexample to the Jacobian Conjecture in dimension three [A26], we write down an explicit endomorphism φ of the third Weyl algebra A₃ that is not an automorphism, refuting the Dixmier Conjecture [D68] for all n ≥ 3. The note is self-contained: every property of the underlying polynomial map that we use — the constant Jacobian, non-injectivity over ℂ and over ℝ, the commutation relations defining φ, and the non-surjectivity of the map itself — is verified by exact symbolic computation in the accompanying script `verify_dixmier.py`, so the correctness of this note does not depend on any unrefereed manuscript. Along the way we record the fiber structure of the map: the x-coordinates of the preimages of a point (A,B,C) satisfy a single cubic **L·x³ + (4−3BC)·x − 2C = 0** whose leading coefficient L = 27A²C² − 18ABC + 16A + B³C − B², and since this cubic has no quadratic term, sheets of the covering can only escape to infinity in pairs; in particular F omits the entire rational curve (B²/12, B, 4/(3B)) and is therefore not surjective, and F is already non-injective on ℝ³ with constant Jacobian, strengthening the situation left by Pinchuk's counterexample [P94] for non-constant Jacobians.

## 1. Introduction

The Dixmier Conjecture (1968) asks whether every endomorphism of the Weyl algebra Aₙ over a field of characteristic zero is an automorphism [D68]. It is classical that the Dixmier Conjecture in dimension n implies the Jacobian Conjecture in dimension n, and conversely the Jacobian Conjecture in dimension 2n implies the Dixmier Conjecture in dimension n [T05, BKK07]; see also [vdE00].

On July 19, 2026, Levent Alpöge announced an explicit polynomial map F: ℂ³ → ℂ³ with constant Jacobian determinant −2 that is generically three-to-one, refuting the Jacobian Conjecture for n = 3 and hence, by adjoining identity coordinates, for all n ≥ 3 [A26]; the map is analyzed in [SBS26]. The purpose of this note is to make the resulting failure of the Dixmier Conjecture *explicit and machine-checkable*: we exhibit the endomorphism φ of A₃ directly, verify its defining relations by exact symbolic computation, and give a short, self-contained proof that φ is not surjective. Since the argument consumes only verified properties of the displayed map — not any unpublished proof — the note stands independently of the peer-review status of [A26].

We work over ℂ throughout; §5 records the real form. Write A₃ for the algebra generated by x₁, x₂, x₃, ∂₁, ∂₂, ∂₃ subject to [xᵢ, xⱼ] = [∂ᵢ, ∂ⱼ] = 0 and [∂ᵢ, xⱼ] = δᵢⱼ, acting faithfully on ℂ[x₁, x₂, x₃] in the usual way.

## 2. The map and its verified properties

Let F = (F₁, F₂, F₃): ℂ³ → ℂ³ be given by

- F₁ = (1 + x₁x₂)³ x₃ + x₂²(1 + x₁x₂)(4 + 3x₁x₂)
- F₂ = x₂ + 3x₁(1 + x₁x₂)² x₃ + 3x₁x₂²(4 + 3x₁x₂)
- F₃ = 2x₁ − 3x₁²x₂ − x₁³x₃

and let J = (∂Fᵢ/∂xⱼ) be its Jacobian matrix.

**Fact 2.1** (script check 1). det J = −2 identically. In particular J⁻¹ = −½·adj(J) is a matrix with *polynomial* entries.

**Fact 2.2** (script check 2). F is not injective: the point (329/270, 19/10, 7/10) has three distinct preimages, namely (1/2, 1/3, 2/5) and the conjugate pair
(−1/4 ± 29√31·i/372, 151/120 ± 21√31·i/40, −3356649/160000 ± 636423√31·i/160000),
each verified by exact substitution.

Both facts are elementary finite computations; the script performs them in exact arithmetic.

## 3. The endomorphism

Let G = J⁻¹ (polynomial by Fact 2.1) and define first-order operators

**Dᵢ = Σₐ Gₐᵢ ∂ₐ  (i = 1, 2, 3),**

with the polynomial coefficients Gₐᵢ standing to the left of ∂ₐ. Explicit entries of G are listed in the Appendix.

**Proposition 3.1** (script check 4). In A₃: [Dᵢ, Fⱼ] = δᵢⱼ and [Dᵢ, Dⱼ] = 0 for all i, j. Together with [Fᵢ, Fⱼ] = 0 this shows that

**φ: A₃ → A₃, xᵢ ↦ Fᵢ, ∂ᵢ ↦ Dᵢ**

is a well-defined algebra endomorphism (universal property of the presentation of A₃). Since A₃ is simple and φ(1) = 1, φ is injective.

*Remarks.* (i) The relation [Dᵢ, Fⱼ] = Σₐ (∂Fⱼ/∂xₐ) Gₐᵢ = (J·G)ⱼᵢ = δᵢⱼ is immediate from G = J⁻¹; the content is [Dᵢ, Dⱼ] = 0, which the script verifies by expanding the nine coefficient identities Σₐ (Gₐᵢ ∂Gᵦⱼ/∂xₐ − Gₐⱼ ∂Gᵦᵢ/∂xₐ) = 0. (ii) Conceptually the Dᵢ are the pullbacks of the coordinate vector fields ∂/∂yᵢ on the target: F is étale, so these commute; but no étaleness argument is needed — the identities are checked directly.

## 4. φ is not an automorphism

**Theorem 4.1.** φ is not surjective; in particular the Dixmier Conjecture fails for A₃, and (by adjoining coordinates xᵢ ↦ xᵢ, ∂ᵢ ↦ ∂ᵢ for i > 3) for Aₙ for every n ≥ 3.

*Proof.* Suppose P ∈ A₃ satisfies φ(P) = x₁. Write P in PBW normal form P = Σ c₍αβ₎ xᵅ ∂ᵝ. Then φ(P) = Σ c₍αβ₎ Fᵅ Dᵝ. Apply both sides, as operators on ℂ[x₁, x₂, x₃], to the constant polynomial 1. Each Dₐ annihilates constants, so every term with |β| ≥ 1 dies, and the terms with β = 0 contribute scalars times Fᵅ · 1:

x₁ = φ(P)(1) = Σ_α c₍α,0₎ Fᵅ = Q₁(F₁, F₂, F₃)

for the polynomial Q₁(y) = Σ_α c₍α,0₎ yᵅ. Already this single identity is impossible: the two preimages displayed in Fact 2.2 have equal F-image but **different first coordinates**, while x₁ = Q₁(F) would force them equal. (Treating all three coordinates the same way would give a full polynomial left inverse Q ∘ F = id, forcing injectivity — the same contradiction with more machinery.) ∎

Note the argument is entirely elementary: no Ax–Grothendieck, no filtration theory — a putative preimage of one coordinate function under φ is *normal-ordered and evaluated at 1*, and out falls a polynomial identity that the displayed collision already refutes.

**Corollary 4.2.** The image φ(A₃) is a proper subalgebra of A₃ isomorphic to A₃: the canonical commutation relations in three degrees of freedom admit a strict self-embedding. (The Kontsevich conjecture, concerning the *automorphism group* of Aₙ, is untouched by this; the failure is confined to non-surjective endomorphisms. Dimensions n = 1, 2 of the Dixmier Conjecture remain open, as does the Jacobian Conjecture in the plane.)

*Remark (the index of the self-embedding).* The generic fiber of F has exactly three points (§5), so the field extension ℂ(x₁,x₂,x₃) ⊇ ℂ(F₁,F₂,F₃) has degree 3: the classical map underlying φ is a degree-3 covering onto its image. The self-embedding φ therefore carries a natural "index 3" — the multiplicity with which A₃ sits over its own copy φ(A₃), matching the three sheets of the crater. Making this module-theoretically precise (and deciding whether 3 is minimal among counterexample-induced endomorphisms) is left open; see OPENINGS.md in the repository root.

## 5. The real form, and the geometry of the fibers

**Fact 5.1** (script check 3). The real point (3/100, 1, 1) has three distinct *real* preimages under F. Hence the restriction F: ℝ³ → ℝ³ — a polynomial local diffeomorphism scaling volume by the constant 2 — is not injective. This strengthens the real Jacobian picture: Pinchuk's counterexample [P94] has nonvanishing but *non-constant* Jacobian; here the Jacobian is constant.

**Fact 5.2** (script check 5). Eliminating x₃ then x₂ from the fiber equations shows that the x₁-coordinate of any preimage of (A, B, C) with C ≠ 0 satisfies

**L·x³ + (4 − 3BC)·x − 2C = 0,  L = 27A²C² − 18ABC + 16A + B³C − B².**

The cubic has no x² term, so when L → 0 two roots escape to infinity simultaneously: fiber cardinalities stratify as 3 (generic), 1 (on the hypersurface L = 0), and 0. Both coefficients L and 4 − 3BC vanish identically on the rational curve

**(A, B, C) = (B²/12, B, 4/(3B)),  B ≠ 0,**

where the cubic degenerates to −2C ≠ 0; hence the fiber there is empty and **F is not surjective**, omitting this entire curve. (Sample points on each stratum are verified exactly in the script; in the slice C = 2/3 the omitted point sits at the cusp of the plane curve L = 0.)
